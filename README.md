# HR-Analytics-Dashboard
## Overview
Developed an HR Analytics Dashboard in Power BI to analyze employee attrition, salary trends, and work-life balance across departments.

## Tools Used - 
* Power BI
* SQL
* DAX
* Power Query

## Key Insights -
* Identified 50.21% attrition rate
* Sales and R&D had the highest attrition
* Analyzed work-life balance and income-level trends

## Features - 
* KPI Dashboard
* Attrition Analysis
* Department-wise Analysis
* Interactive Visualizations
